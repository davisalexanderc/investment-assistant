# Security & Safety Guardrails

## Prompt/agent safeguards
- Detect direct and indirect prompt injection.
- Block attempts to reveal system prompts/secrets/API keys.
- Detect and flag requests for direct personalized investment advice.

## Output safeguards
OutputVerificationAgent checks:
- unsupported claims
- hallucinated metrics
- buy/sell language
- overconfident predictions
- missing provenance
- unsafe links/markdown

## Financial constraints
- Research/education only.
- No investment recommendations.
